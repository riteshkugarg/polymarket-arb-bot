## Performance Optimization

The filter is optimized for early rejection:

1. **Layer 1** (Time-Horizon): Fastest check - regex search
2. **Layer 2** (Binary): Quick JSON parse
3. **Layer 3** (Status): Simple boolean checks
4. **Layer 4-8**: Only evaluated if market passes first 3 layers

**Average rejection time**: <1ms per market  
**Scan 200 markets**: ~200ms total
